下载依赖

本文中相关操作系统及依赖包的版本如下：

  • centos-release-7-4.1708.el7.centos.x86_64
  • openldap-clients-2.4.44-5.el7.x86_64：包含客户端程序，用来访问和修改 OpenLDAP 目录
  • openldap-servers-2.4.44-5.el7.x86_64：包含主 LDAP 服务器 slapd 和同步服务器 slurpd 服务器、迁移脚本和相关文件
